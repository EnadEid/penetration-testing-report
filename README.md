**# Penetration Testing Project — Kioptrix Level 1 \& Mr-Robot 1**!\[Security](https://img.shields.io/badge/Type-Penetration%20Testing-red)!\[Status](https://img.shields.io/badge/Status-Completed-brightgreen)!\[Course](https://img.shields.io/badge/Course-Ethical%20Hacking%20605445-blue)**## Overview**A full black-box penetration test conducted on two intentionally vulnerablevirtual machines as part of Ethical Hacking course 605445 at the Universityof Petra. The engagement follows the Penetration Testing Execution Standard(PTES) methodology and documents the complete attack lifecycle fromreconnaissance to root access.\---**## Target Systems**| **Target** | **Type** | **Difficulty** | **Result** ||---|---|---|---|| Kioptrix Level 1 | Linux / Network Services | Beginner | Rooted || Mr-Robot 1 | Linux / Web Application | Intermediate | Rooted |\---**## Attack Summary****### Kioptrix Level 1**\- Discovered outdated Samba 2.2.x on port 139\- Exploited CVE-2003-0201 trans2open buffer overflow via Metasploit\- Obtained unauthenticated root shell\- Retrieved /etc/shadow confirming full system compromise**### Mr-Robot 1**\- Discovered WordPress 4.3.1 with exposed wordlist via robots.txt\- Enumerated username via WPScan\- Brute forced admin credentials via XML-RPC interface\- Injected PHP reverse shell via WordPress theme editor\- Cracked MD5 hash to escalate to robot user\- Exploited SUID nmap binary to escalate to root\- Retrieved all 3 keys confirming full system compromise\---**## Vulnerabilities Found**| #
